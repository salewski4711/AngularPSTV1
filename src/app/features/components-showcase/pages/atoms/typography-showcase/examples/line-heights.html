Relaxed Line Height (leading-relaxed)
Body text benefits from relaxed line height for improved readability. When reading longer paragraphs, the extra space between lines helps the eye track from the end of one line to the beginning of the next. This is especially important for content-heavy interfaces where users need to read and comprehend information quickly.
Best for body text and longer content
